## Compiling

```bash
# Clone this repository
$ git clone https://github.com/quiet-visage/FieldFusion field_fusion

# Go into the repository
$ cd field_fusion

# Create build directory
$ mkdir -p build

# Generate build files
$ cd build; cmake ..

# Compile
$ make -j8
```
